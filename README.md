# Projetos do 1º Semestre de engenharia de computação 

##  Descrição

Este projeto foi desenvolvido para expor as experiências que tive no primeiro semestre de engenharia de computação com a linguagem c++.

## 🛠️ Tecnologias utilizadas

* C++
* Git e GitHub

##  Como executar

1. Clone o repositório:

```bash
git clone https://github.com/seuusuario/seurepositorio.git
```

2. Entre na pasta:

```bash
cd seurepositorio
```

3. Compile (caso seja C++):

```bash
g++ arquivo.cpp -o programa
```

4. Execute:

```bash
./programa
```

## 📚 Aprendizados

* Aprendi sobre lógica de programação
* Uso de estruturas como if, switch, loops
* Versionamento com Git
